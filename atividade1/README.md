### **Explicações**:

- #### A parte 1 do trabalho está no arquivo **filecopy.c**;
- #### **Para compilar e executar**:
- ```
    gcc filecopy.c -o filecopy -std=c99 -Wall
    ./filecopy arquivo1 arquivo2
  ```

- #### A parte 2 é o arquivo **treecopy.c** que necessita do arquivo **arquivo.c**;
- - #### **Para compilar e executar**:
    ```
      gcc treecopy.c -o treecopy -std=c99 -Wall
      ./treecopy diretorio1 diretorio2
    ```
